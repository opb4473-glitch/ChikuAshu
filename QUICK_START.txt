SETUP (5 MINUTES):

1. CREATE MONGODB DATABASE
   ├─ Go to: https://www.mongodb.com/cloud/atlas
   ├─ Sign up for FREE account
   ├─ Create new project
   ├─ Create FREE M0 cluster (takes 2-3 minutes)
   └─ Wait for cluster to fully deploy

2. GET CONNECTION STRING
   ├─ Click "Connect" on your cluster
   ├─ Select "Drivers"
   ├─ Copy the full connection string:
   │  Example: mongodb+srv://user:pass@cluster.mongodb.net/?retryWrites=true...
   └─ Save it temporarily

3. WHITELIST YOUR IP
   ├─ In MongoDB left sidebar: "Network Access"
   ├─ Click "Add IP Address"
   ├─ Click "Allow Access from Anywhere"
   └─ Confirm

4. ADD TO VERCEL PROJECT
   ├─ Click settings ⚙️ (top right of this page)
   ├─ Go to "Vars" tab
   ├─ Click "Add Variable"
   ├─ Name: MONGODB_URI
   ├─ Value: [paste your connection string]
   ├─ Click "Save"
   └─ Refresh the chat app

5. INITIALIZE DATABASE
   ├─ Click "Need to Setup Database?" on login page
   ├─ Follow the in-app setup steps
   ├─ Click "Initialize Database" button
   └─ Wait for confirmation

6. LOGIN!
   ├─ Ashu: username "ashu" + password "ashu123"
   ├─ Chiku: username "chiku" + password "chiku123"
   └─ You'll stay logged in even after refreshing!

TROUBLESHOOTING:

ERROR: "MONGODB_URI is not set"
SOLUTION:
  • Check you added variable in Vercel Vars
  • Make sure you copied FULL connection string
  • Refresh the page after adding variable

ERROR: "Authentication failed"
SOLUTION:
  • Check MongoDB username/password in connection string
  • Whitelist your IP in Network Access
  • Try "Allow Access from Anywhere" temporarily

ERROR: "Cannot connect"
SOLUTION:
  • Wait for cluster to fully deploy (2-3 minutes)
  • Verify connection string in MongoDB Atlas
  • Test connection string in MongoDB Compass

ERROR: "Users already exist"
SOLUTION:
  • Just login with the default credentials!
  • Or clear the database in MongoDB Atlas
