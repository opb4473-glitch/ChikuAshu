🚀 ONE-TIME SETUP (First Time Only)
═══════════════════════════════════════════════════════════════════════════════

PHASE 1: MongoDB Setup (3-4 minutes)
  ☐ Go to https://www.mongodb.com/cloud/atlas
  ☐ Click "Start Free" button
  ☐ Sign up with email
  ☐ Verify email
  ☐ Create project (any name, choose FREE tier)
  ☐ Create cluster (M0 Sandbox, FREE)
  ☐ Wait for cluster to deploy (2-3 minutes)
  ☐ Click "Connect" on your cluster
  ☐ Select "Drivers" tab
  ☐ Copy connection string (starts with "mongodb+srv://")
  ☐ Go to "Network Access" → Add IP → "Allow from Anywhere"
  ☐ Confirm


PHASE 2: Vercel Configuration (2-3 minutes)
  ☐ You're already viewing this chat app on Vercel
  ☐ Click settings icon ⚙️ (top right of screen)
  ☐ Go to "Vars" tab
  ☐ Click "Add Variable"
  ☐ Name: MONGODB_URI (exactly!)
  ☐ Value: Paste your MongoDB connection string
  ☐ Click "Save"
  ☐ Refresh the chat app page (Ctrl+Shift+R)


PHASE 3: Initialize Database (1 minute)
  ☐ Refresh the chat app page
  ☐ Click "Need to Setup Database?" link
  ☐ Follow the in-app instructions
  ☐ Click "Initialize Database" button
  ☐ Wait for confirmation message
  ☐ Your Ashu and Chiku accounts are ready!


PHASE 4: First Login (30 seconds)
  ☐ Enter username: ashu
  ☐ Enter password: ashu123
  ☐ Click "Enter Our Chat"
  ☐ You're in the chat!


PHASE 5: Invite Chiku (30 seconds)
  ☐ Share this same URL with Chiku
  ☐ She clicks the link
  ☐ She follows same setup (or just logs in if already done)
  ☐ She enters username: chiku
  ☐ She enters password: chiku123
  ☐ She's in the chat!

TOTAL TIME: ~10 minutes (first time only)

❓ COMMON QUESTIONS
═══════════════════════════════════════════════════════════════════════════════

Q: Do I need to create an account?
A: No! Ashu and Chiku accounts are created automatically.

Q: Can I change my password?
A: Not via UI, but you can modify the setup script if needed.

Q: Will my messages expire?
A: No! They stay forever unless you delete them.

Q: What if I clear my browser history?
A: Your session cookie might clear. Just login again with same password.

Q: Can Chiku see when I login?
A: No. There are no notifications or "seen" indicators.

Q: What happens if MongoDB goes down?
A: You can't send/receive messages. But it's very reliable!

Q: Can I add more users?
A: Yes, by modifying the setup script.

Q: What if I lose the URL?
A: Get it from Vercel dashboard or ask Chiku who has it.

Q: Is this app always free?
A: Yes! MongoDB free tier and Vercel free tier are unlimited for this use.

Q: Can you access my messages?
A: No. I don't have access to your MongoDB database.

Q: What if I want to reset everything?
A: Delete the MongoDB database and run init again.


🆘 IF SOMETHING GOES WRONG
═══════════════════════════════════════════════════════════════════════════════

ERROR: "MONGODB_URI is not set"
SOLUTION:
  1. Check you added MONGODB_URI in Vercel Vars
  2. Make sure you copied the FULL connection string
  3. Refresh the page (Ctrl+Shift+R)
  4. Click "Initialize Database" again

ERROR: "Cannot connect to database"
SOLUTION:
  1. Check MongoDB cluster status (might still deploying)
  2. Verify connection string is correct
  3. Check IP is whitelisted in Network Access
  4. Wait 2-3 minutes for cluster to fully initialize

ERROR: "Invalid credentials"
SOLUTION:
  1. Use username: ashu and password: ashu123
  2. Or use username: chiku and password: chiku123
  3. Check you're using exact spelling
  4. Try initializing database again

ERROR: "Messages not loading"
SOLUTION:
  1. Click Refresh button to reload messages
  2. Check MongoDB is still connected
  3. Refresh entire page (Ctrl+Shift+R)
  4. Check internet connection
