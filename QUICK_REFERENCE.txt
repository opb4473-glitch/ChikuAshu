🚀 3-STEP SETUP
═══════════════════════════════════════════════════════════════════════════════

STEP 1: MONGODB
  → mongodb.com/cloud/atlas
  → Create FREE cluster
  → Copy connection string

STEP 2: VERCEL VARS
  → Click settings ⚙️ (top right)
  → Go to "Vars" tab
  → Add: MONGODB_URI = [connection string]

STEP 3: INITIALIZE
  → Click "Initialize Database" on login page
  → Done!

🆘 COMMON FIXES
═══════════════════════════════════════════════════════════════════════════════

ERROR: "MONGODB_URI is not set"
FIX: 
  1. Check you added it in Vercel Vars
  2. Refresh page (Ctrl+Shift+R)
  3. Try Initialize again

ERROR: "Cannot connect"
FIX:
  1. Check MongoDB cluster deployed (wait 2-3 min)
  2. Check connection string is correct
  3. Check IP whitelisted

ERROR: "Login failed"
FIX:
  1. Use: ashu / ashu123 or chiku / chiku123
  2. Try Initialize Database again
  3. Check caps lock
