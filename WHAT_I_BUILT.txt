🚀 HOW TO START (3 SIMPLE STEPS)
═══════════════════════════════════════════════════════════════════════════════

STEP 1: GET MONGODB (3 minutes)
  1. Go to https://www.mongodb.com/cloud/atlas
  2. Sign up FREE
  3. Create FREE cluster (M0)
  4. Copy connection string
  5. Whitelist your IP

STEP 2: CONFIGURE VERCEL (1 minute)
  1. Click settings ⚙️ (top right of this app)
  2. Go to "Vars" tab
  3. Add MONGODB_URI = [your connection string]
  4. Save and refresh

STEP 3: INITIALIZE & CHAT (1 minute)
  1. Click "Need to Setup Database?" on login page
  2. Click "Initialize Database"
  3. Login with ashu/ashu123
  4. Share URL with Chiku
  5. Start chatting!

🎯 HOW IT WORKS
═══════════════════════════════════════════════════════════════════════════════

SENDING A MESSAGE:
  1. You type "Hey Chiku!"
  2. Click Send
  3. Message goes to /api/messages via POST
  4. Saved in MongoDB with your username
  5. You see it appear immediately

RECEIVING A MESSAGE:
  1. Chiku sends "Hey Ashu!"
  2. It's stored in MongoDB
  3. You click Refresh button
  4. App loads all messages via GET /api/messages
  5. You see Chiku's message appear

NO REAL-TIME LOGIC:
  ✓ No WebSockets needed
  ✓ No background connections
  ✓ Perfect for sharing links
  ✓ Simple and reliable
  ✓ Lower server cost
  ✓ No notifications
