🚀 QUICK START (3 STEPS)
═══════════════════════════════════════════════════════════════════════════════

STEP 1: GET MONGODB CONNECTION STRING
  1. Go to https://www.mongodb.com/cloud/atlas
  2. Sign up FREE
  3. Create FREE M0 cluster
  4. Get connection string
  5. Whitelist your IP
  
STEP 2: ADD TO VERCEL VARS
  1. Click settings ⚙️ (top right)
  2. Go to "Vars" tab
  3. Add: MONGODB_URI = [connection string]
  4. Save

STEP 3: INITIALIZE & LOGIN
  1. Refresh this page
  2. Click "Need to Setup Database?"
  3. Click "Initialize Database"
  4. Login with ashu/ashu123
  5. Share URL with Chiku!
